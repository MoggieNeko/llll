Instagram 影片收集分析器

不再假裝可匿名精準全網搜 IG。這版改成真正可用流程：先在 Instagram 搜尋，再把可見 Reel / Post 收集回來，交給網站分析與排序。

建議工作流
  1. 先去 Instagram 站內搜尋，例如 hashtag、Explore、Reels、指定帳號。
  2. 在該頁按 Bookmarklet / Chrome Extension，收集可見貼文連結和簡介片段。
  3. 貼回這個網站，再按語言、產品字眼、背景主色做篩選與排序。

Bookmarklet 用法
  1. 將 browser_helper/bookmarklet.txt 內容存成瀏覽器書籤。
  2. 打開 Instagram 搜尋結果頁、Reels 頁、hashtag 頁或指定帳號頁。
  3. 按一下書籤，會自動複製 JSON 收集結果到剪貼簿。
  4. 貼回這裡分析。
